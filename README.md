# SFCANet V4: Advanced DeepFake Detection

This repository contains the Phase 4 implementation of the SFCANet (Spatial-Frequency Cross-Attention Network) architecture for deepfake detection. 

## Overview
Phase 4 focuses on achieving state-of-the-art (SOTA) cross-domain generalization using:
- **Spatial Branch:** ConvNeXt-Tiny backbone.
- **Frequency Branch:** Multi-Scale DCT CNN with 1x1 pointwise bottleneck convolutions.
- **Fusion:** Adaptive Frequency-Guided Attention (AFGA).

## Current Status
- Initializing cross-dataset validation experiments.
- Implementing Median-based Segment-Aware Pooling to mitigate burst noise.
